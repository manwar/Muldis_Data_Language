=head1 DESCRIPTION

This document defines the Muldis D Plain Text package
C<Muldis_D::Plain_Text:"http://muldis.com":"0.200.1"> which provides the
reference implementations of parsers and generators for Muldis D Plain Text
source code, mainly in the form of functions that map Muldis D C<Text>
values containing such with Muldis D C<Package> values.

Note that for brevity this document will use the unqualified term
I<argument> to refer to a function's conceptual argument that is actually
an attribute of a I<FDN__Tuple_Inf> typed actual argument; the latter will
be qualified as I<topic> in the rarer circumstance where that is meant.

=head1 PACKAGES

    Muldis_D:Plain_Text:"http://muldis.com":"0.200.1";
    script ASCII;
    package
    {
        identity MDPT ::= Muldis_D::Plain_Text:"http://muldis.com":"0.200.1";
        foundation "http://muldis.com":"0.200.1";
        uses MD ::= Muldis_D:"http://muldis.com":"0.200.1";
        floating {::MD, ::MD::Source_Code, ::MDPT};
        ...
    };
