=head1 DESCRIPTION

This document defines the Muldis D Plain Text package
C<Muldis_D:"http://muldis.com":"0.200.1"> which provides the common core
system-defined data types and operators that regular users of the language
would employ directly in their applications and schemas.

=head1 MAXIMAL AND MINIMAL DATA TYPES

=head2 Universal

    Muldis_D::Universal ::= type default false;

The enumeration contract C<Universal> represents the infinite I<universal
type>, which is the maximal data type of the entire Muldis D type system
and consists of all values which can possibly exist.  It also represents
the infinite Muldis D Foundation type I<FDN__Any_Thing>.  Its default value
is C<false>.

=head2 Empty

    Muldis_D::Empty ::= type of {};

The enumeration contract C<Empty> represents the finite I<empty type>,
which is the minimal data type of the entire Muldis D type system and
consists of exactly zero values.  It can not have any default value.

=head1 GENERIC SIMPLE DATA TYPES

=head2 Boolean

    Muldis_D::Boolean ::= type
        where topic -->^ FDN__isa_2VL_Bool
        default false
        composes {Boolish and_provides_its_default,
            Orderable and_provides_its_default};

The enumeration contract C<Boolean> represents the finite Muldis D
Foundation type I<FDN__2VL_Bool>.  A C<Boolean> value is a general purpose
2-valued logic boolean, or specifically it is one of the 2 values C<false>
and C<true>.  Its default value is C<false>.

        where topic -->^ FDN__isa_Big_Int

The enumeration contract C<Integer> represents the infinite Muldis D
Foundation type I<FDN__Big_Int>.  An C<Integer> value is a general purpose
